How we expand
Each interest below becomes its own new ad set. Same offer and same creative as the control set, with only the interest layer changing. Interest sizes and IDs were validated live in Meta on June 2, 2026.

One interest, one ad set

Build a separate ad set per interest layer. Do not stack the interests onto an existing set, and do not layer them on top of the lookalikes. The point is to read each interest cleanly.

Event level, not venue level

Meta campaigns target the whole market per event, so the new ad sets sit at the event campaign level. There is no venue-level targeting to expand into.

Funded by reallocation

Pull the budget for new ad sets from the underperforming ad sets on the same campaign. This is a swap, not an incremental budget request.

Read after 5 to 7 days

Compare each interest ad set against the control on cost per lead. Keep the winners, cut the rest. Inside a 7-day read window the interest layer is the fastest lever available.

When to deploy — the 14-day lever

Any event with fewer than 14 days to close and running under registration target on Meta is a trigger to spin up interest ad sets. Creative and messaging tests need 7 to 10 days to gather signal plus a few more to react, so inside the 14-day window the interest swap is the only lever fast enough to move volume. Build checklist
  1. Pick the event and open its Meta campaign. Confirm the event is under target and identify the weakest ad sets to defund.
  2. Create one new ad set per Tier 1 interest. Same offer, same creative as the control, only the interest layer changes.
  3. Name each ad set [MKT]_INT_[Interest], for example DCA_INT_RealEstateInvesting, so reads stay clean.
  4. Set each new ad set budget by pulling from the underperforming ad sets on the same campaign. No incremental budget.
  5. Start with the top three to five Tier 1 interests rather than all ten, so each gets enough budget to exit learning.
  6. Read at 5 to 7 days on cost per lead versus control. Keep the winners, cut the rest, and only then test Tier 2 with a narrow.
